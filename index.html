Full-stack development

Enhance your JavaScript skills to become proficient with both front-end and back-end development using Promises, Classes, and ES6+ language features.

Hands-on Learning

Build REST compliant APIs for web and mobile using Node.js and industry standard tools, like Express, JWT (javascript web tokens), Jest, Cloud Services, and Passport.js for authentication.
